Playback

Output device
Choose which audio device TuneBridge plays through. You must set this here — macOS per-app routing (System Settings › Sound) does not work when bit-perfect mode is active, because TuneBridge holds exclusive access to the device.
Bit-perfect output BP
CoreAudio exclusive mode: bypasses the macOS system mixer so your FLAC files play without resampling or volume scaling. Produces the purest signal path — recommended for DAC/amp setups. Silences system audio while active.
Player BP badge appears only when:
  • mpv backend is available
  • Bit-perfect output is enabled
  • Current track source is lossless
  • EQ/PEQ is disabled
  • Volume is 100% and not muted

Help Center

Need quick steps for playlist export, import, and DAP sync? Start here.

Quick Start
  1. Create or open a playlist in TuneBridge.
  2. Choose a connected device from the export controls in the playlist header.
  3. If no device is connected, use Download M3U and copy it to your device's configured export folder.
  4. Run a library rescan on your DAP if the playlist does not appear immediately.
TuneBridge exports playlists as .m3u for broad DAP compatibility.
