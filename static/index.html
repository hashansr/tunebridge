Help & Guide

Using Playlists on Poweramp (FiiO M21)
  1. Select a playlist and click Poweramp (M3U) to download the file.
  2. Connect your FiiO M21 to your Mac — it mounts at /Volumes/FIIO M21/
  3. Copy the .m3u file to FIIO M21 › Playlists ›
  4. Eject the device, open Poweramp → Library → Playlists.
  5. If the playlist doesn't appear, go to Poweramp Settings → Library → Rescan.
Playlist name shows with an extension? Poweramp uses the filename as the playlist display name. Playlists are now exported as .m3u which Poweramp strips from the display name automatically. If you have old .m3u8 files on your device, delete them and re-export.
Tip: When the FiiO M21 is connected, a device button appears on the playlist page that copies the file directly.
Using Playlists on AP80 Pro Max
  1. One-time setup: On the AP80, create and save one playlist using the heart/star icon during playback. This makes the device create the playlist_data/ folder on the SD card.
  2. Connect your AP80 to your Mac via USB (Settings → USB Mode → Storage) — it mounts at /Volumes/AP80/
  3. Export using AP80 Pro Max (M3U) to download, or click → AP80 to copy directly when the device is connected.
  4. If copying manually, place the .m3u file in playlist_data/ at the SD card root (not in Music/ or a Playlists/ folder).
  5. On the AP80: Music → ☆ → Playlists → Load Playlist.
Playlist shows 0 songs? This happens if you created the playlist_data/ folder manually on your Mac. Delete the folder, let the AP80 recreate it by saving a dummy playlist, then re-copy your M3U files.
Can't see playlist_data/ from your Mac? It may be hidden. Use the → AP80 direct copy button — it creates the folder automatically.
How playlist paths work

Playlist path formats differ by device:

Poweramp (FiiO M21) — relative paths from the SD card root:

Music/Artist/Album/01. Track Title.flac

AP80 Pro Max — relative paths from playlist_data/ (one level above Music/):

../Music/Artist/Album/01. Track Title.flac
Importing existing playlists

Click the ↑ (upload) button next to Playlists in the sidebar to import any .m3u or .m3u8 file.

The app matches tracks by file path and falls back to artist + title matching. Tracks not found in your library are shown as unmatched so you can review them.

Music source & library

Your library is scanned from /Volumes/Storage/Music/FLAC/. Click the ↺ button at the bottom of the sidebar to rescan after adding new music.

Device Settings

Poweramp — FiiO M21

The internal path is prepended to track paths in the exported M3U8, e.g. /storage/sdcard0/Music/…. Leave empty to use relative paths.

AP80 Pro Max

Playlists are written to playlist_data/ at the SD card root using relative paths — no additional configuration needed.
