Help Center

Need quick steps for playlist export, import, and DAP sync? Start here.

Quick Start
  1. Create or open a playlist in TuneBridge.
  2. Choose a connected device from the export controls in the playlist header.
  3. If no device is connected, use Download M3U and copy it to your device's configured export folder.
  4. Run a library rescan on your DAP if the playlist does not appear immediately.
TuneBridge exports playlists as .m3u for broad DAP compatibility.

TuneBridge — Privacy Policy

TuneBridge runs entirely on your Mac. No data is ever sent to the
developer.

WHAT WE COLLECT
Nothing. TuneBridge does not collect, store, or transmit any personal
data to external servers operated by the developer.

NETWORK REQUESTS
Network requests only happen when you initiate them:
  • squig.link — when you import IEM frequency response measurements
  • Last.fm — when you provide your own API key for artist images
  • Fanart.tv — when you provide your own API key for artist images

TuneBridge sends no additional personal information alongside these
requests beyond what is necessary to fetch the data you asked for.

THIRD-PARTY SERVICES
Last.fm, Fanart.tv, and squig.link have their own privacy policies.
Please review them before providing API keys. TuneBridge has no
control over how these services handle the data you send them.

DATA STORAGE
All your data (playlists, settings, listening history) is stored
locally on this device in:
  ~/Library/Application Support/TuneBridge/
